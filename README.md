# C# OOP Einstieg

**Dozent:** Tom Selig (BitLC)  
**Thema:** Einstieg in C# und Objektorientierte Programmierung (OOP)

## Projektbeschreibung
Dieses Repository dient als begleitendes Material für den C#-Kurs. Es behandelt die Grundlagen der Programmierung sowie fortgeschrittene OOP-Konzepte. Das Projekt ist modular aufgebaut und wird über einen zentralen **Controller** gesteuert.

## Struktur & Navigation
Das Projekt besteht aus nummerierten Aufgaben ("Units"), die aufeinander aufbauen:

- **Aufgabe_00_Intro**: Projektbeschreibung & KI-Kontext.
- **Aufgabe_01_Array**: Grundlagen zu Arrays.
- **Aufgabe_02_TV**: Einführung in Klassen (TV-Simulation).
- **Aufgabe_03_Konstruktoren**: Vertiefung zu Konstruktoren.
- ... und weitere fortgeschrittene Themen (Vererbung, Beziehungen).

## Hinweise für Prüfer (IHK)
Der Code ist ausführlich kommentiert (`///` Summaries), um die verwendeten Konzepte und die Funktionsweise auch für Dritte verständlich zu machen. UML-Diagramme liegen den jeweiligen Modulen bei.
