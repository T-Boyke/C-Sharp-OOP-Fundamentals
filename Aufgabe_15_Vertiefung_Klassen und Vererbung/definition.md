**Klasse**
Eine Klasse ist der Bauplan oder die Schablone für ein Objekt. Sie definiert die Attribute (Eigenschaften) und Methoden (Verhalten), die alle Instanzen dieser Klasse gemeinsam haben.

**Vererbung**
Vererbung ist ein Mechanismus, bei dem eine Unterklasse die Eigenschaften und Methoden einer Oberklasse übernimmt. Dies ermöglicht die Wiederverwendung von Code und die Abbildung von "ist-ein"-Beziehungen (Spezialisierung).

**Objekt**
Ein Objekt ist eine konkrete Instanz einer Klasse, die zur Laufzeit existiert. Es besitzt konkrete Werte für die im Bauplan (Klasse) definierten Attribute.
